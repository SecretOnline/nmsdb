NMSDB

The NMSDB is a project to catalogue as much about No Man's Sky as possible

Due to the game being unreleased, there's nothing catalogued about the actual game world, or the players who are in it

However, we do have an Information Repository that holds almost everything we know so far from press events and other media coverage
